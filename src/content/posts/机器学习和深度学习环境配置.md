# conda虚拟环境
## 创建虚拟环境
```shell
conda create -n py310 python=3.10
```
用版本比较新的库就创建3.12版本
```shell
conda create -n py312 python=3.12
```
同 conda create --name gpu_default python=3.12

安装完之后查看pytorh版本
```shell
python -V
```
或`Python --version`和什么指令一样，全面Python忽略大小写，但是后面V要严格一致

## 激活虚拟环境
```shell
conda activate py310
```

## uv
```
pip install uv
```
导入uv之后可以用uv pip install指令加速下载

# 深度学习环境

## PyTorch版本

### CV
```shell
# ROCM 6.0 (Linux only)
pip install torch==2.3.1 torchvision==0.18.1 torchaudio==2.3.1 --index-url https://download.pytorch.org/whl/rocm6.0
# CUDA 11.8
pip install torch==2.3.1 torchvision==0.18.1 torchaudio==2.3.1 --index-url https://download.pytorch.org/whl/cu118
# CUDA 12.1
pip install torch==2.3.1 torchvision==0.18.1 torchaudio==2.3.1 --index-url https://download.pytorch.org/whl/cu121
# CPU only
pip install torch==2.3.1 torchvision==0.18.1 torchaudio==2.3.1 --index-url https://download.pytorch.org/whl/cpu
```

cpu流程：
```shell
uv pip install torch==2.3.1 torchvision==0.18.1 torchaudio==2.3.1
```
然后可以在终端运行进行测试：
```shell
# 测试安装是否正常
# 1. 进入python命令⾏
python
# 2. python命令⾏中输入⼀下命令
import torch
torch.__version__
torch.rand(2,3)
# 查看正常的numpy 版本
import numpy as np
np.__version__
```



GPU流程：

```shell
uv pip install torch==2.3.1 torchvision==0.18.1 torchaudio==2.3.1 --index-url https://download.pytorch.org/whl/cu121
```

测试：

```shell
# 测试安装是否正常
# 1. 进入python命令⾏
python
# 2. python命令⾏中输入⼀下命令
import torch
torch.__version__
torch.rand(2,3)
torch.rand(2,3).cuda()
torch.rand(2,3).to(device=torch.device('cuda'))
torch.cuda.is_available()
torch.cuda.device_count()
# 查看正常的numpy 版本
import numpy as np
np.__version__
```
