# 感知机模型实现流程

感知机是一种二分类线性模型，用于寻找一个能够将正负样本分开的超平面。模型的分类函数为：

$$f(x) = sign(w \cdot x + b)$$

其中：

- $w$：权重向量
- $b$：偏置项
- $x$：输入样本
- $sign$：符号函数，输出类别$+1$ 或 $-1$

## 1. 准备训练数据

示例中使用二维样本点进行二分类：

```python
import numpy as np
import matplotlib.pyplot as plt

p_x = np.array([
    [4.0, 2.0],
    [3.0, 2.0],
    [2.5, 1.0],
    [2.0, 1.0]
])

y = np.array([1, 1, -1, -1])
```

其中，`p_x` 表示训练样本，`y` 表示样本标签。

## 2. 可视化样本点

为了观察数据分布，可以将正样本和负样本分别画出：

```
for i in range(len(p_x)):
    if y[i] == 1:
        plt.plot(p_x[i][0], p_x[i][1], 'ro')
    else:
        plt.plot(p_x[i][0], p_x[i][1], 'bo')

plt.show()
```

红色点表示正类，蓝色点表示负类。

## 3. 初始化模型参数

感知机训练前，需要初始化权重、偏置和学习率：

```
w = np.array([0.4, 0.4])
b = 0.5

alpha = 0.1
```

其中 `alpha` 是学习率，用来控制每次参数更新的步长。

## 4. 判断误分类点

对于样本 $(x_i, y_i)$，如果模型预测值与真实标签不一致，则该点为误分类点：

$$ y_i \neq \operatorname{sign}\left(w \cdot x_i + b\right) $$

也可以写成数学形式：

$$y_i(w \cdot x_i + b) \leq 0$$

当上述条件成立时，说明当前分类超平面没有正确分类该样本。

## 5. 更新参数

感知机的核心思想是：找到误分类点后，利用该样本更新参数，使分类超平面向正确方向移动。

更新公式为：

$$
\begin{aligned}
w &= w + \alpha\,y_i x_i \\
b &= b + \alpha\,y_i
\end{aligned}
$$

代码实现如下：

```
for i in range(100):
    choice = -1

    for j in range(len(p_x)):
        if y[j] != np.sign(np.dot(w, p_x[j]) + b):
            choice = j
            break

    if choice == -1:
        break

    w = w + alpha * y[choice] * p_x[choice]
    b = b + alpha * y[choice]

print(i)
print('w:{}\nb:{}'.format(w, b))
```

这里使用的是随机梯度下降思想，即每次找到一个误分类点后立即更新参数。

训练结果为：

```
31
w:[-0.15  0.6 ]
b:-0.4
```

说明模型经过若干次迭代后，找到了可以划分样本的参数。

## 6. 绘制分类直线

二维情况下，分类边界满足：

$$
w_1 x_1 + w_2 x_2 + b = 0
$$

可以将其变形为：

$$
w_1 x_1 + w_2 x_2 + b = 0
$$

代码如下：

```
line_x = [0, 10]
line_y = [0, 0]

for i in range(len(line_x)):
    if w[1] != 0:
        line_y[i] = (-w[0] * line_x[i] - b) / w[1]
    else:
        line_x = [-b / w[0], -b / w[0]]
        line_y = [0, 1]

plt.plot(line_x, line_y)
plt.show()
```

如果 $w_2 \neq 0$，可以直接根据公式计算直线上的点；如果 $w_2 = 0$，说明分类边界是一条垂直线。

## 7. 使用模型进行分类

训练完成后，感知机模型可以通过以下公式进行预测：

```
model = np.sign(np.dot(w, x) + b)
```

如果输出为 `1`，表示样本属于正类；如果输出为 `-1`，表示样本属于负类。

## 总结

感知机模型的实现流程可以概括为：

1. 初始化权重 `w` 和偏置 `b`
2. 遍历训练数据，寻找误分类点
3. 使用误分类点更新参数
4. 重复迭代，直到没有误分类点或达到最大迭代次数
5. 使用最终的 `w` 和 `b` 得到分类超平面

感知机适用于线性可分数据。当数据线性可分时，算法可以通过不断修正误分类点，最终得到一个能够正确分类训练样本的线性分类器。
