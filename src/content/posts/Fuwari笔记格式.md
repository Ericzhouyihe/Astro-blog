# Front-matter格式
## 常用

```markdown
---
title: 我的新文章
published: 2026-07-20
description: ''
image: ''
tags: []
category: ''
draft: false	 # false=发布，true=草稿
lang: ''		# 指定单篇文章的语言代码（如 zh-CN、en、ja 等）

hidden: false # 是否在列表隐藏
summary: 文章摘要，不写会自动截取正文
pinned: false	# 是否置顶
slug: "BP"  # 直接地控制URL
---
```
