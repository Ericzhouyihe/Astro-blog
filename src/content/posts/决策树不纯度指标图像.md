# 决策树不纯度指标图像

在决策树分类模型中，每一次划分特征时，都需要衡量当前节点的“纯度”。

如果一个节点中的样本几乎都属于同一类别，说明这个节点比较纯；如果不同类别的样本混在一起，说明这个节点不纯。

常见的不纯度指标有：

- 信息熵
- Gini 系数
- 错误率

本文通过绘制图像，对比这三个指标在二分类场景下的变化规律。

import numpy as np

## 二、信息熵

信息熵用于衡量数据的不确定性。

对于二分类问题，假设某一类样本比例为 $p$，另一类样本比例为 $1-p$，则信息熵为：

$ H(p)= -p\log_2 p - (1-p)\log_2(1-p) $

代码实现如下：

```python
def H(p):
    return np.sum([-i * np.log2(i) for i in p if i != 0])
```

这里需要注意，当 $p=0$ 时，$p\log p$ 按极限处理为 0，所以代码中排除了 `i == 0` 的情况。

信息熵的特点是：

- 当 $p=0$ 或 $p=1$ 时，节点完全纯，信息熵为 0
- 当 $p=0.5$ 时，两类样本各占一半，不确定性最大，信息熵最大
- 二分类情况下，信息熵最大值为 1

## 三、Gini 系数

Gini 系数是 CART 决策树中常用的不纯度指标。

公式为：$ Gini(p)=1-\sum_{k=1}^{K}p_k^2 $

对于二分类问题：$ Gini(p)=1-p^2-(1-p)^2 $

化简可得：$ Gini(p)=2p(1-p) $

代码实现如下：

```python
def gini(p):
    return 1 - np.sum([i * i for i in p])
```

Gini 系数的特点是：

- 当 $p=0$ 或 $p=1$ 时，节点完全纯，Gini 为 0
- 当 $p=0.5$ 时，节点最混乱，Gini 最大
- 二分类情况下，Gini 最大值为 0.5

## 四、错误率

错误率可以理解为：如果当前节点直接预测为样本最多的类别，那么会有多少比例的样本被预测错误。

公式为：$Error(p)=1-\max(p_k)$

对于二分类问题：$Error(p)=1-\max(p,1-p)$

也可以写成：$Error(p)=\min(p,1-p)$

代码实现如下：

```python
def error(p):
    return 1 - np.max(p)
```

错误率的特点是：

- 当 $p=0$ 或 $p=1$ 时，错误率为 0
- 当 $p=0.5$ 时，错误率最大
- 二分类情况下，错误率最大值为 0.5
- 相比信息熵和 Gini，错误率变化不够敏感

## 五、绘制不纯度指标图像

在二分类问题中，只需要让其中一类的概率 $p$ 从 0 变化到 1，另一类的概率就是 $1-p$。

```python
P0 = np.linspace(0.0, 1.0, 1000)
P1 = 1 - P0
```

然后分别计算三种不纯度指标：

```python
y1 = [H([p0, 1 - p0]) * 0.5 for p0 in P0]
y2 = [gini([p0, 1 - p0]) for p0 in P0]
y3 = [error([p0, 1 - p0]) for p0 in P0]
```

这里对信息熵乘以了 `0.5`，是因为二分类信息熵的最大值是 1，而 Gini 和错误率的最大值都是 0.5。

为了方便放在同一张图中对比，将信息熵缩放为“熵之半”。

完整绘图代码如下：

```python
plt.figure(num="不纯度")

P0 = np.linspace(0.0, 1.0, 1000)
P1 = 1 - P0

y1 = [H([p0, 1 - p0]) * 0.5 for p0 in P0]
y2 = [gini([p0, 1 - p0]) for p0 in P0]
y3 = [error([p0, 1 - p0]) for p0 in P0]

plt.plot(P0, y1, label=u'熵之半')
plt.plot(P0, y2, label=u'gini')
plt.plot(P0, y3, label=u'error')

plt.legend(loc='upper left')
plt.xlabel(u'p', fontsize=18)
plt.ylabel(u'不纯度', fontsize=18)
plt.title(u'不纯度指标', fontsize=20)
plt.show()
```

## 六、图像规律分析

从图像上可以看出，三种指标都有类似的整体趋势：

1. 当 $p=0$ 时，节点中只有一类样本，不纯度为 0。
2. 当 $p=1$ 时，节点中也只有一类样本，不纯度为 0。
3. 当 $p=0.5$ 时，两类样本各占一半，不纯度最大。
4. 曲线关于 $p=0.5$ 对称。

也就是说，不管使用信息熵、Gini 系数还是错误率，它们都认为：$p=0.5$时节点最不纯。

而：$ p=0 \quad 或 \quad p=1 $时节点最纯。

## 七、熵和熵之半

notebook 中还单独绘制了信息熵和熵之半的对比图：

```python
plt.figure(num="熵之半")

y = [H([p0, 1 - p0]) for p0 in P0]
y1 = [H([p0, 1 - p0]) * 0.5 for p0 in P0]

plt.plot(P0, y, label=u'H(X)')
plt.plot(P0, y1, label=u'熵之半')

plt.legend(loc='upper left')
plt.xlabel(u'p', fontsize=18)
plt.ylabel(u'熵', fontsize=18)
plt.title(u'熵和熵之半', fontsize=20)
plt.show()
```

可以看到：

- $H(X)$ 是原始信息熵
- 熵之半就是 $0.5H(X)$
- 两条曲线形状完全一致，只是纵轴数值不同

所以“熵之半”并不是新的不纯度指标，只是为了和 Gini、错误率放在同一张图中比较，做了一个缩放处理。

## 八、三种指标对比

| 指标      | 公式                 | 最大值位置 | 二分类最大值 | 常见算法   |
| --------- | -------------------- | ---------- | ------------ | ---------- |
| 信息熵    | $-\sum p_k\log_2p_k$ | $p=0.5$    | 1            | ID3、C4.5  |
| Gini 系数 | $1-\sum p_k^2$       | $p=0.5$    | 0.5          | CART       |
| 错误率    | $1-\max(p_k)$        | $p=0.5$    | 0.5          | 剪枝、评估 |

三者都可以衡量节点的不纯度，但敏感程度不同：

- 信息熵对概率变化最敏感，计算时需要对数运算。
- Gini 系数计算简单，效果稳定，是 CART 中常用的划分标准。
- 错误率最直观，但对节点纯度变化不够敏感，一般较少直接用于划分特征。

## 九、小结

本节通过图像对比了信息熵、Gini 系数和错误率三种不纯度指标。

它们的共同点是：

- 节点越纯，指标值越小。
- 节点越混杂，指标值越大。
- 二分类中，当两类样本比例接近 $0.5:0.5$ 时，不纯度最大。

在决策树中，选择划分特征的核心就是让划分后的节点尽可能变纯。也就是让划分后的不纯度尽可能降低，从而得到信息增益或 Gini 增益更大的特征。
